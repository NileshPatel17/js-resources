## set email local to the current directory

```
git config user.email "your_email@abc.com"
```
### Confirm that you have set your email address:
```
git config --global user.email
```
## set email globally

```
git config --global user.email "your_email@abc.com"
```
### Confirm that you have set your email address:
```
git config --global user.email
```
